# Student Score Predictor

A Machine Learning web application built using **Python**, **Scikit-learn**, and **Streamlit** to predict student math scores based on study habits and attendance.

---

# Features

- Predicts student scores using Machine Learning
- Interactive Streamlit web interface
- User-friendly sliders for input
- Real-time prediction results
- Beginner-friendly Data Science project

---

# Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
